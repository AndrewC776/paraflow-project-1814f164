# Soft Float - Gradient
## Colors
- **Primary Color**: 
  - **color-primary-base**:`linear-gradient(76.799576deg, #1B99FF 1%, #64B4F5 55%, #B2DAFF 92%);` - For surface
  - **color-primary-Lighter**: #CFE4FC - For surface
  - **color-primary-darker**: #508FE0 - For special/emphasized text
- **color-accent**: #FFFFCC - For special/emphasized tag and lable surface 
- **color-page-background**: `radial-gradient(117% 55% at 50% 32%, #F6FAFE 47%, #E9EFFB 100%);`
- **surface**:
  - **color-surface-primary**: #FAFDFF - For cards
  - **color-surface-secondary**: #EBF6FF - For cards
  - **color-primary-tertiary**: #FFFFFFE6
  - **color-surface-insert**: #D5E1F5B3
 
- **Text and Icon Colors**: 
  - **color-text-primary**: #111827 - A soft black for primary text.
  - **color-text-secondary**: #6B7280 
  - **color-text-tertiary**: #9CA3AF
  - **color-text-on-dark**: #FFFFFF - For text on dark or primary-base backgrounds.

- **Functional Colors**: 
  - Used for status indicators and notifications. Use sparingly to maintain a clean aesthetic.
  - **color-success**: #91E2AC
  - **color-warning**: #F5A897


## Typography
- **Font Stack**: 
  - **font-family-base**: `-apple-system, BlinkMacSystemFont, "Segoe UI"` — For regular UI copy

- **Font Size & Weight**: 
  - **Caption**: 10px / 400
  - **Body**: 12px / 400
  - **Card Title / Emphasized Text**: 14px / 500
  - **Page Title**: 20px / 500
  - **Headline**: 30px / 500
  - **Display**: 40px / 500

- **Line Height**: 1.4

## Border Radius
  - rounded-xl - (12px) For buttons, inputs, and small containers
  - rounded-2xl - (16px) For cards, main containers, and content blocks
  - rounded-full - For tags, pills, bottom navigation, and circular selection states

## Layout & Spacing
- **Spacing Scale**: 
  - **Base Unit**: 8px
  - **spacing-tight**: 8px - For close-related elements.
  - **spacing-compact**: 12px - For list items and small gaps.
  - **spacing-standard**: 16px - For general padding, margins and section/module spacing.
  - **spacing-comfortable**: 24px - For page padding

## Borders
- No borders.

## Dividers
- No dividers or separator lines.

## Shadows & Effects
- No shadows.

## Assets
### Image
  - Always use `object-cover` for proper image scaling and cropping.

### Logo
- To protect copyright, do **NOT** use real product logos as a logo for a new product, individual user, or other company products.
- **Icon-based**:
  - **Graphic**: Use a simple, relevant Lucide icon from Iconify (e.g., `lucide:calendar` for a scheduler, `lucide:home` for a smart home app).

### Icon
- Use Lucide icons from Iconify.
- To ensure aesthetic layout, each icon should be centered in a square container matching the icon's size. This container is only for layout optimization and does not imply the icon has a background; other properties should be determined based on actual needs.
- Use Tailwind font size to control icon size
- Example:
  ```html
  <div class="flex items-center justify-center bg-transparent w-4 h-4">
  <iconify-icon icon="lucide:flag" class="text-base"></iconify-icon>
  </div>
  ```

    <!-- Main content goes here, use section with horizontal page padding(px-6) -->
